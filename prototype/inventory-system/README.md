# 家庭用在庫管理システム

家庭内の食料品や日用品の在庫を管理するWebアプリケーションです。

## 機能

### 実装済み機能
- ✅ ユーザー登録・ログイン
- ✅ 商品の登録・編集・削除
- ✅ カテゴリ管理
- ✅ 在庫数の管理（購入・使用）
- ✅ 在庫履歴の記録
- ✅ ダッシュボード（統計情報・在庫アラート）
- ✅ レスポンシブデザイン（スマホ対応）

### 今後実装予定
- ⏳ 賞味期限管理
- ⏳ バーコード読み取り機能
- ⏳ 在庫アラート通知
- ⏳ 買い物リスト自動生成

## セットアップ

### 前提条件
- Docker
- Docker Compose

### インストール手順

1. リポジトリのクローン（またはファイルのダウンロード）

2. Docker環境の起動
```bash
cd inventory-system
docker-compose up -d
```

3. アプリケーションへのアクセス
- フロントエンド: http://localhost:3000
- バックエンドAPI: http://localhost:8000
- APIドキュメント: http://localhost:8000/docs

### 初回セットアップ

1. ブラウザで http://localhost:3000 にアクセス
2. 「新規登録はこちら」から最初のユーザーを作成
3. ログイン後、カテゴリを追加（例：調味料、飲料、日用品など）
4. 商品を登録
5. 在庫管理を開始

## 使い方

### カテゴリの追加
1. 「商品管理」ページへ移動
2. 「カテゴリ追加」ボタンをクリック
3. カテゴリ名を入力して追加

### 商品の登録
1. 「商品管理」ページへ移動
2. 「商品追加」ボタンをクリック
3. 商品情報を入力
   - 商品名
   - カテゴリ
   - バーコード（任意）
   - 単位（個/本/袋/ml/g/kg/L）

### 在庫の管理
1. 「在庫管理」ページへ移動
2. 商品ごとに「購入」または「使用」ボタンをクリック
3. 数量を入力して確定

### ダッシュボード
- 全体の統計情報を表示
- 在庫が5個以下の商品を警告表示

### 履歴の確認
- 「履歴」ページで過去の在庫増減を確認可能

## 開発

### バックエンドの開発
```bash
cd backend
# 仮想環境の作成
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 依存パッケージのインストール
pip install -r requirements.txt

# 開発サーバーの起動
uvicorn main:app --reload
```

### フロントエンドの開発
```bash
cd frontend
# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev
```

### データベースの場所
- SQLiteデータベースは `backend/data/inventory.db` に保存されます
- Dockerボリューム `inventory-data` にマウントされているため、コンテナを再起動してもデータは保持されます

## トラブルシューティング

### ポートが既に使用されている
```bash
# 使用中のポートを確認
docker-compose ps

# コンテナを停止
docker-compose down

# ポート番号を変更する場合は docker-compose.yml を編集
```

### データをリセットしたい
```bash
# コンテナとボリュームを削除
docker-compose down -v

# 再起動
docker-compose up -d
```

### フロントエンドが表示されない
```bash
# フロントエンドのログを確認
docker-compose logs frontend

# コンテナを再ビルド
docker-compose up -d --build frontend
```
